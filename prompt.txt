Steamlit Cloud で実行できる新たなアプリを作成したい。仕様概要を以下に記述する。わからないことや、作成にあたり必要なことがあれば事前に確認してください。

・DXF Extract Labels と DXF Visual Diff の機能を利用する
・DXF Extract Labels の機能を利用して以下を行う
	・複数のDXFファイルを一括してアップロートする
	・アップロードされたDXFファイルから図面番号を抽出して、「図番」と「流用元図番」ペアのリストを作成し、表示する
	・アップロードされたDXFファイルの中に「図番」に対応する「流用元図面」のDXFファイルがない「図番」一覧を表示し、改めてDXFファイルのアップロードを促す
	・「追加アップロード」のボタン押下で、再度DXFファイルのアップロードを行い、「図番」と「流用元図番」ペアのリストをアップデートし、表示する。
・「比較開始」のボタン押下で、DXF Visual Diff の機能を離礁して以下を実行する
	・DXF Visual Diff のファイルペアの指定は省略し、「オプション設定」「オフセット補正設定（オプション）」はそのまま有効にする
	・ペアリストにもとづいた差分のDXFファイルを作成する


元プロジェクトの DXF Extract Labels が正しく機能していないファイルがあります。
DXF-extract_labels ディレクトリ下にある EE6321-039-06A.dxf は、図番が EE6321-039-06A で、流用元図番が EE6097-039-06C なのですが、正しく抽出できていません。1ファイルに複数の図面が描かれているからだと考えられます。このような場合でも正しく流用元図番を抽出できるようにしたい。
まずは、DXF Extract Labels プロジェクトをアップデートしたい。


図番と流用元図番の表示位置には以下の関係があります。
・図番はファイル名と一致している
・流用元図番は図番が表示されている位置の左上にある
・図番には「DWG No.」（空白部分は開業）のラベルが左側の近くにある
・流用元図番には「流用元図番」のラベルがすぐ近くの左側にある
